# Editing guide

The website now follows the final webpage guideline sections:

- Abstract
- Introduction
- Related work
- Method
- Experiments
- Conclusion and limitations
- Individual contributions

## Files you will usually edit

- `index.html`: section text, links, authors, contribution statements.
- `static/data/results.json`: numeric metrics used by the table and quality chart.
- `static/images/`: exported figures.
